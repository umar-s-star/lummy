## 🚀 Быстрый старт

```bash
npm install
npm run dev
```

Сборка:
```bash
npm run build
npm run preview

## 🗺 Yandex Maps (Branches)

- Подключение через Yandex Maps API v2.1 (бесплатно без ключа для dev)
- Кастомные брендированные маркеры с логотипом Lummy
- Кластеризация при большом числе точек
- Красивые всплывающие карточки (balloon) с полной информацией
- Кнопки «Позвонить» и «Маршрут» прямо на карте
- Панорамирование при выборе филиала из сайдбара
- Glassmorphism-оверлей с брендингом

> **Для production:** добавьте ваш API-ключ Яндекс в `BranchesPage.tsx`:
> ```ts
> const YMAPS_SRC = 'https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ&lang=ru_RU';
> ```

## 🔧 Административная панель `/admin`

Полностью рабочий фронтенд с mock-данными, готовый к подключению backend.

### Разделы:
- **Дашборд** — метрики, статистика, последние заказы
- **Категории** — полный CRUD с модальными окнами, toggle активности
- **Блюда** — CRUD, поиск, фильтр по категории, пагинация, скрыть/показать
- **Заказы** — список с фильтром статусов, смена статуса inline
- **Отзывы** — модерация: одобрить / скрыть / удалить
- **Филиалы** — CRUD с координатами
- **Настройки** — конфигурация сайта

### Особенности:
- Модальные окна с валидацией форм
- Подтверждение удаления через диалог
- Лоадеры, empty states, toast-уведомления
- Пагинация в таблице блюд
- Мобильная нижняя навигация

## 🔌 Подключение Backend

Архитектура готова к API-интеграции:

1. Создайте `src/api/client.ts` (axios instance с JWT interceptors)
2. Создайте `src/api/dishes.ts`, `src/api/orders.ts` и т.д.
3. Оберните вызовы в `@tanstack/react-query` (уже в зависимостях)
4. Замените `useState(INIT_DATA)` в компонентах на `useQuery`/`useMutation`
